# The Void 👻

Anonymous real-time messaging for friend groups.

## Setup

1. **Supabase**: Go to [supabase.com](https://supabase.com), create a free project
2. **Database**: Go to SQL Editor → paste the contents of `supabase/migration.sql` → Run
3. **Config**: `.env` is already set with the Supabase credentials
4. **Install**: `npm install`
5. **Run**: `npm run dev`
6. **Open**: http://localhost:5173

## Features
- 💬 Anonymous messaging — zero identity
- 🎭 Confessions with purple glow
- 📊 Live polls with real-time votes
- 💣 Self-destructing messages with countdown
- ⏱️ Ticking bombs (timed group prompts)
- 🔥 Daily streak counter
- 🌡️ Room vibe meter (chaotic / chill / wild / deep / unhinged)
- 🏆 Wall of Fame (messages with 5+ reactions)
- 💎 Rare diamond reactions (10% chance)
- 👻 Ghost presence indicators + typing indicator
- ↩️ Reply threads
- 📌 Pinned messages
- 🌙 Dark & light theme toggle
- 📱 Mobile-first responsive design
